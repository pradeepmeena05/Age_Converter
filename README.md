# Age Converter

This is an Age Converter application that allows users to convert their age in years into other units such as months, days, hours, minutes, and seconds.

## Features

- Convert age from years to months, days, hours, minutes, and seconds.
- Easy-to-use interface.
- Option to enter birthdate or directly input the age.

## Getting Started

To run the Age Converter application locally, follow these steps:

### Prerequisites

- **Android Studio**
- **Java** or **Kotlin**

### Installation

1. Clone the repository:
   git clone https://github.com/pradeepmeena05/Age_Converter.git
   
3. Open the project in Android Studio.
4. Run the app on an emulator or connected Android device.

#Usage
1. Launch the app on your device.
2. Enter your birthdate, or input the number of years you'd like to convert.
3. The app will display your age in months, days, hours, minutes, and seconds.
4. Optionally, you can reset the input to try with different values.
